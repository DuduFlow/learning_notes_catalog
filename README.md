## 檔案結構

```
learning_notes_catalog/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── main.js
├── .nojekyll
└── README.md
```

## 已上架連結

- 能力地圖分類
- 關鍵字搜尋

- 依月份自動分組
- 課程資料集中在 `js/main.js` 的 `courses` 陣列，後續新增筆記連結只要更新該陣列
