## 功能

- 能力地圖分類
- 關鍵字搜尋
- 狀態篩選
- 依月份自動分組
- 課程資料集中在 `js/main.js` 的 `courses` 陣列，後續新增筆記連結只要更新該陣列
- 已加入跨瀏覽器與裝置 favicon：一般瀏覽器、Apple Touch Icon、Android/Chrome Web App、Windows tile
